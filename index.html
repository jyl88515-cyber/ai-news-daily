手动更新设置

"手动更新" 按钮会调用 GitHub Actions 重新抓取。首次使用需要粘贴一个 GitHub Personal Access Token（PAT），仅存在你本地浏览器（localStorage），不会上传。

  1. 打开 Fine-grained tokens 点 Generate new token
  2. Repository access：只勾选本仓库
  3. Permissions → Repository → Actions: Read and write
  4. 生成后复制 token 粘到下面
